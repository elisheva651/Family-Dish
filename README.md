# Family Dish

A mobile-first web app (PWA) for groups of people to share recipes together. Families, friends, and communities can create private groups, share their favorite recipes, and comment on each other's dishes.

## Features

- Google/Apple one-tap sign-in
- Create and join recipe-sharing groups with invite codes
- Add recipes with photos, ingredients, instructions, and tags
- Browse recipes by person or by category
- Comment on recipes with text and photos
- Manager-controlled group membership and moderation
- Works on phones — add to home screen for an app-like experience

## Tech Stack

- React (PWA)
- Firebase Authentication, Firestore, Storage, Hosting
